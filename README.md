# Cajero Automático

Este es un proyecto simple de cajero automático implementado en TypeScript.

## Instalación

```bash
npm install
```

## Compilación

```bash
npm run build
```

## Ejecución

```bash
npm start
```

## Arquitectura

- `src/models/CuentaBancaria.ts`: Clase principal para la cuenta bancaria.
- `src/services/CajeroService.ts`: Lógica de negocio del cajero.
- `src/utils/validaciones.ts`: Funciones de validación.
- `src/index.ts`: Punto de entrada con pruebas básicas.
